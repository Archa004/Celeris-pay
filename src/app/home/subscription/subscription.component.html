Subscription API

Partner PSPs can use this API to send a recurring payment transaction to be processed by CelerisPay’s payment gateway. This API functions as a client-side POST, either with form data (utilizing CelerisPay’s provided data wrappers) or a JSON object.

The Payment API supports the options of (1) using the Partner’s own payment page or (2) using CelerisPay’s hosted payment page. Recurring Payment is only available for selected payment methods like Credit Cards and Sepa.
